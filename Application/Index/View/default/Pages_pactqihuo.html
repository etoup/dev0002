2.1 各方同意并确认，各方通过自行或授权有关方根据六合资本网站有关规则和说明，在六合资本网站进行配资申请和投标操作等方式确认签署本协议。

2.3 各方通过上述方式接受本协议且丙方审核通过时，本协议立即成立；协议成立的同时甲乙双方不可撤销地授权丙方将出借款项在扣除相关费用后划转至相应证券（期货）账户；出借款项由丙方根据授权划转完成时，本协议正式生效。

3.1 与丙方合作的证券（期货）公司开设的指定账户为出借账户，乙方有权使用账户中的资金总额按本协议约定进行中国国内四个期货交易所（上海，大连，郑州，中金所）所有“活跃交易品种”（“活跃交易品种”是指前三个交易日平均成交高于20000手的主力期货合约）的主力合约的交易。

3.2 为保证资金安全，乙方将资金转入出借账户内，作为借款保证金。以六合资本网站或银行的转账凭证作为保证金的凭证。甲方的借款和乙方的保证金在以上证券交易账户内以用于证券(期货）投资。

3.3 甲乙双方共同同意委托丙方对协议中的证券（期货）交易账户及资金进行全程管理和风险控制。

3.4 出借账户中的证券（期货）交易账户的交易密码应当乙丙双方知悉，在合同期内非特殊原因，任何一方均不得擅自更改，否则视为严重违约，按账户资金总额的10%计算违约金。

3.5 特殊免责条款：鉴于计算机软硬件环境的差异性和复杂性，及协议过程中不能预见，不能避免，不能克服的不可抗力影响，丙方基于网络平台所提供的服务可能因政策原因、网络原因、市场波动、乙方或第三方业务系统故障等非甲方丙方原因出现停顿、延误或中断，乙方对此给予充分的谅解，不对此追究丙方之法律和经济责任。

4. 各方权利和义务

4.1 甲方的权利和义务

4.1.1 甲方有权代理出借人出借资金、签订《借款协议》和《借款确认书》，按本协议约定向借款人出借资金、收回借款和收取利息、综合管理费、违约金，及时将期货账户的交易密码告知乙方，并保密相关交易信息，不得透露期货账户的交易详情,不参与期货交易账户的盈利分配。

4.1.2 甲方及委托单位有权监管期货账户和期货子账户的运作，对乙方进行账户管理和风险控制。

4.1.3 除本协议另有约定外，甲方在本协议执行期内不得提取、划转账户内的资金，不得修改交易密码，不得转托管，不得进行任何买卖期货的操作，不得撤销指定交易，不得注销、挂失或停止使用期货账户、期货子账户（因期货公司原因除外），也不得调出或处置期货账户内资产；否则，给乙方造成损失的，甲方应承担相应赔偿责任。

4.1.4 借款期满，乙方应将借款本金归还给甲方并不再使用甲方期货账户进行交易，否则甲方有权销户或修改交易密码，调出或处置期货账户内资产。

4.2 乙方权利和义务

4.2.1 乙方可以通过甲方期货账户按照本协议约定进行买卖操作，有权收取使用账户内资金总额投资所得的收益，承担全部损失；无论账户盈亏如何，乙方须按本协议约定向甲方支付利息和综合管理费。

4.2.2 如任一账户内的所有交易品种出售（即清盘）后，还不足以归还该笔借款本金金额，乙方必须在清盘后三日内将亏损额补足归还给甲方，否则甲方有权进行追偿或从乙方进行交易的其他账户中扣除相应金额。

4.2.3 乙方操作账户时，仅可购买约定的期货品种或主力合约，甲方及委托单位有权对违反本约定的品种立即进行部分或全部平仓处理，乙方认可甲方及委托单位的此项操作，并对因此带来的损失承担完全的责任。

4.2.4 乙方不得进行坐庄、对敲、接盘、内幕信息、开设聚赌场所等违反期货交易法律法规及期货公司规定的交易，如有任一次上述交易行为，应支付给甲方借款额10%的违约金，并直接从保证金中扣除，且乙方不得转借帐户给他人操作，因上述行为产生的一切责任和损失，均由乙方承担。

4.2.5 如因乙方违规操作、债务等原因，导致甲方账户（包括银行账户和期货账户）被依法采取查封、冻结、划拨等强制措施，造成甲方损失的，乙方应从账户被依法处置之日起按借款本金及利息管理费之和的0.1%每日向甲方支付违约金，如违约金不足以弥补给甲方造成的损失，乙方应承担赔偿责任。借款期满仍未解除强制措施的,本协议自动终止,乙方应另行全额归还甲方出借资金、支付利息和综合管理费，并按约支付违约金。

4.2.6 过夜商品期货总仓位不得超过保证金余额的二倍；过夜股指期货总仓位不能超过保证金余额的三倍，所有不符合规定的持仓在四大交易所规定交易时间收盘前5分钟之前自行调整好仓位，且这个时间段乙方只能平仓，不得再开新仓。如果乙方违反上述约定操作，甲方及委托单位有权对乙方所有持仓品种进行部分或全部平仓处理，造成的一切损失均由乙方承担。如果甲方及委托单位已经实行平仓，乙方再建立仓位或因此被迫留仓，此次交易如产生亏损，亏损额由乙方负责；如产生盈利，盈利部分归甲方所有。乙方承诺对因此带来的损失承担全部责任。

4.2.7 对于接近涨跌停板附近的交易，合约价格2%以内，不得新开与持有反向单，甲方有权对违规持仓按市价强行平仓。如乙方在甲方完成平仓后再次建仓，属于乙方严重违约，甲方可即时终止合约，因此产生的亏损，由乙方完全负责，且账户内全部资金归甲方所有。

4.2.8 乙方在操作过程中，由于操作不当导致亏损时，应严格遵循警戒线和平仓线的规则。

（1）警戒线：乙方保证期货账户内的自有保证金不得低于初始保证金的 50%，当期货账户自有保证金低于警戒线时，乙方必须补入保证金，使期货账户自有保证金达到警戒线以上，如未及时追加保证金且自有保证金低于平仓线时，甲方及委托单位有权将期货子帐户平仓。甲方有权但无义务对警戒线进行任何形式的通知。

（2）平仓线：当期货账户内的自有保证金低于初始保证金的 30% 时，甲方及委托单位有权在不通知乙方的前提下将持仓品种全部平仓并限制开仓。当期货帐户因行情涨跌幅达到极限标准即涨跌停板（+/-）10%的商品或股指只能以自有资金持仓,否则甲方及委托单位有权对期货账户强制全部平仓，由此而造成的一切损失均由乙方承担。

4.2.9 提取盈利：当指定账户资产总值高于双方共同注资额的盈利部分，乙方有权要求甲方或丙方协助在两个工作日内提取至乙方的六合资本平台账户或其绑定银行卡内。

4.2.10 乙方若要续约，必须在离借款到期的三个交易日前，向丙方提出续约申请的相关事宜。乙方正常终止协议，应在借款到期的前一个交易日，将期货账户内的总资产全部变为可提现的货币资金，未及时清仓或期满后又表示不续约的，需支付一个月利息和管理费作为违约金。

4.2.11 乙方应确保其提供的信息和资料的真实性，不得提供虚假信息或隐瞒重要事实。乙方不得将本协议项下的任何权利义务转让给任何其他方。

4.2.12 乙方签订本合同后，表明乙方已完全了解网上交易可能存在的相应风险，以及自身决策失误造成的损失，并承诺自行承担。
