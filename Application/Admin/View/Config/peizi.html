  • 配资相关参数设置
配资设置
最大配资比例 倍
最大配资比例根据实际情况适当调整
投标周期 天
默认投标周期，过期标的将回收
月利率 %
基础月利率，根据实际情况适当调整
基准利率 %
基准利率设置，和业务提成有关，必须小于月利率
年化收益 %
请填写大于1的数值，最多两位小数，该值和投资收益有关
意向金比率 %
请根据实际情况调整，必须小于1的小数
延期服务费率 %
请根据实际情况调整，必须小于1的小数
预警线 %
设置预警线比例值，大于100
平仓线 %
设置平仓线比例值，大于100
配资方积分基数 %
配资方配资成功的积分计算基数，计算方式：配资金额*配资方基数/12*配资期限，填0或者不填将不做积分处理
投资方积分基数 %
投资方投资成功的积分计算基数，计算方式：投资金额*投资方基数/12*投资期限，填0或者不填将不做积分处理
推广收益率 %
根据配资、投资金额计算推广收益，计算方式：配资金额*投资收益率，填0或者不填将不做处理
期货最大配资比例 倍
期货最大配资比例根据实际情况适当调整
期货月利率 %
基础月利率，根据实际情况适当调整
期货预警线 %
设置预警线比例值，大于100
期货平仓线 %
设置平仓线比例值，大于100
