4. 各方权利和义务

4.1 甲方的权利和义务

4.1.1 甲方是该资金的合法所有人，如果第三人对资金归属、合法性问题发生争议，由甲方负责解决。如甲方未能解决，则放弃享有其出借款项所带来的利息收益。

4.1.2 甲方享有其所出借款项所带来的利息收益，并应主动缴纳由利息所得带来的可能的税费。

4.1.3 在乙方操作股票交易过程中，遇到所购买的股票出现停牌、摘牌等不可预见情况而超出合同约定的时间时，甲方应同意按本协议的相关约定延长借款时间。

4.1.4 如根据本协议约定的顺序清偿时，乙方还款不足以偿还约定本金、利息及逾期罚息等款项的，甲方同意各自按照其借出款项比例收取还款。

4.1.5 就甲方在本协议项下对乙方的部分或全部债权，甲方有权根据自己的意愿对外转让，并授权丙方可以视情况（必要时）将债权转让事项通知乙方。

4.1.6 如乙方违约，甲方有权要求丙方提供其已获得的乙方信息。

4.1.7 甲方承诺对依据本协议获得的乙方信息或资料予以保密，除用于本协议目的进行出借与合理催收外，不得向外转让或披露。

4.2 乙方权利和义务

4.2.1 乙方必须按期足额向甲方归还每期应还本金和利息，向丙方按期足额支付相关借款管理费和服务费用，并支付与乙方逾期及提前还款有关的费用。

4.2.2 乙方承诺所借款项不用于任何违法用途。乙方自协议生效起，可以使用股票账户内的资金购买上海和深圳证券交易所交易的股票，盈亏均由乙方自行承担。

4.2.3 乙方购买的股票原则上应业绩优良、流通股本规模适度、流动性好。乙方购买的股票要按照以下条件：

（1） 不得购买或持有S、ST、*ST、S*ST、SST以及被交易所特别处理的股票；

（2）不得购买期权、权证类可以T+0交易的证券；

（3）不得购买首日上市新股（或复牌首日的股票）等当日涨跌幅不受10%限制的股票；

（4）不得进行新股、增发股票、可转债的网上或网下申购；

（5）不得私自开通沪港通，或进行沪港通股票的交易；

（6）不得购买封闭式基金，理财产品等赎回变现期限超过一个月的有价证券；

（7）不得购买或持有证监会发布的退市公告、风险揭示提示及被查的相关股票；

（8）不得存在坐庄、对敲、接盘、内幕信息等违反股票交易法律及证券公司规定的交易；

（9）单只股票的市值不得超过流通总市值的5%，购买创业板不能超过账户总资产的30%，购买主板单只股票不得超过账户总资产的60%。

若乙方从事上述交易，甲方或丙方有权立即强制平仓及限制交易，乙方需支付其借款金额的10%作为违约金，直接从保证金中扣除，由此产生的一切损失均由乙方承担。

4.2.4 在乙方进行股票交易过程中，遇到所购买的股票出现停牌、摘牌等不可预见情况时，乙方应立即另行支付所持停牌股票市值一倍的保证金到出借账户内作为风险保证金，若超出合同约定的时间时，乙方应立即通过丙方同甲方协商，在甲方同意的前提下按本协议的相关约定延长借款时间。若因特殊原因无法延期，乙方应立即归还甲方出资的本金，被停牌或摘牌的股票属于乙方所有，乙方应支付占用甲方账户的费用，费用另行协商。若乙方不同意，甲方和丙方有权立即解除合约，账户内所持有股票及剩余资金全部归甲方和丙方所有。

4.2.5 乙方在操作过程中，由于操作不当导致亏损时，应严格遵循警戒线和平仓线的规则。

（1）警戒线：出借账户内总资产低于甲方出借资金总额的115%时，丙方有权关闭出借账户开仓权限，乙方应补充保证金至出借账户内的资金总额的115%以上。若乙方因客观原因无法及时追加保证金，应立即减仓，将仓位控制在剩余保证金的5倍以内。否则，甲方和丙方有权限制交易或强制减仓，无需征得乙方同意。甲方或丙方对警戒线有权但无义务进行通知。

（2）平仓线：当出借账户内总资产低于甲方出借资金的110%时，甲方或丙方有权在低于110% 的任何可能成交的价位强行平仓，卖出出借账户中的所持股票，乙方同时失去对出借账户的操作权，甲方或丙方有权在卖出后从出借账户中提取本金、利息及相应违约金。乙方同意甲方或丙方进行上述证券交易和资金提取，不得以任何理由抗辩。甲方或丙方对平仓线有权但无义务进行通知。

如出借账户中金额不足支付甲方出借资金及乙方应付账户使用费的，乙方应在三个交易日内补足差额。逾期未补足的，乙方应按借款金额的0.1%/日支付违约金直至本金全部补足时止。甲方和丙方对差额部分具有追偿权。

4.2.6 提取盈利：当指定机构账户资产总值高于双方共同注资额的盈利部分，乙方有权要求甲方或丙方协助在一个月内提取至乙方的六合资本平台账户或其绑定银行卡内。盈利中高于账户总资金110%以上部分方可提取，并按千元的整数倍提取。每月盈利提取次数为1次。

4.2.7 乙方若要续约，必须在离借款到期的三个交易日前，向丙方提出续约申请的相关事宜。乙方正常终止协议，应在借款到期的前一个交易日，将股票账户内的总资产全部变为可提现的货币资金，未及时清仓或期满后又表示不续约的，需支付一个月利息和管理费作为违约金。

4.2.8 乙方确认甲方可以根据自己的意愿对其根据本协议形成的对乙方的部分或全部债权进行转让，并特此同意甲方届时转让债权无需另行通知乙方；同时乙方承诺若届时需要，将采取必要措施配合债权的一次或多次转让或届时债权人债权的实现。在债权转让后，甲方在本协议项下的权利和义务自动转移到债权受让人名下，乙方应按照本协议的约定向债权受让人支付每期应还借款本息，不得以未接到债权转让通知为由拒绝履行还款义务。

（5）拖欠的本金；

（6）拖欠丙方的借款管理费；

（7）正常的利息；

（8）正常的本金；

（9）丙方的借款管理费。

罚息总额 = 逾期本息总额×对应罚息利率×逾期天数
逾期天数 1—7天 7天及以上
罚息利率 0.1%/天 0.3%/天
